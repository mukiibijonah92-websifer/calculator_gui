# Java Swing Calculator GUI

A simple, interactive desktop calculator application built using Java Swing and the NetBeans Palette drag-and-drop GUI builder.

#  Features
* Standard arithmetic operations: Addition, Subtraction, Multiplication, and Division.
* Responsive Grid/Absolute layout designed via NetBeans GUI Palette.
* Clear (`C`) button to reset the calculator display.
* Decimal point support for floating-point calculations.

# Built With
* **Java Development Kit (JDK)** - Version 8 or higher
* **NetBeans IDE** - Used for Swing GUI layout design
